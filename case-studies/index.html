Proyek pilihan dengan dampak nyata.

Kumpulan pilihan sistem yang saya rancang dan kirimkan — mulai dari platform manajemen risiko enterprise hingga transformasi CRM bertenaga AI. Masing-masing mengikuti alur yang sama: realitas berantakan → sistem yang dirancang → produk yang berjalan.

PT Naska Jala Dewa
Salesforce-grade CRM Transformation
n8n Vertex AI WhatsApp API Odoo PHP Docker
Masalah

CRM Odoo standar tidak punya intelijen lead berbasis AI, penegakan SLA real-time, dan visibilitas eksekutif — sehingga tiket terlewat dan siklus deal melambat.

Solusi

Membangun 18+ workflow n8n di sekitar Odoo: engine AI Lead Scoring, eskalasi SLA 3 tingkat, WhatsApp Executive Dashboard, otomasi survei CSAT, dan AP Portal khusus untuk ingest invoice.

Dampak

Dirancang untuk menghilangkan tiket terlewat lewat eskalasi bertingkat, memberi eksekutif visibilitas deal real-time di WhatsApp, dan menggantikan kerja helpdesk manual dengan agen AI dwi-mode bernama "Aina".

PT Adhi Karya
E-Risk — Enterprise Risk Management
Python Odoo 16 PostgreSQL REST API SSO
Masalah

Proses risiko enterprise terpecah-pecah di spreadsheet dan tool yang tidak terhubung, tanpa visibilitas real-time terhadap KPI, treatment, maupun penilaian dampak.

Solusi

Merancang 30+ modul Odoo 16 kustom — risk register, dashboard interaktif, pelacakan KPI, penilaian dampak, realisasi treatment, validasi tier, SSO, dan REST API publik.

Dampak

Dibangun menjadi satu sumber kebenaran untuk risiko enterprise — dengan workflow persetujuan, tema dark mode, dan integrasi mulus ke platform PT. Adhi Karya.

30+ modul Risk register Dashboard KPI SSO & REST API

PT Kripto Syariah Indonesia
CryptoSharia — Learning & Membership Ecosystem
SvelteKit TypeScript Tailwind 4 PostgreSQL Drizzle ORM OpenAPI 3.1
Masalah

Indonesia belum punya platform edukasi kripto yang kredibel dan sesuai syariah, dengan kedalaman e-learning sekaligus sistem screening token yang dapat diverifikasi.

Solusi

Membangun platform pembelajaran dan keanggotaan terstruktur untuk ekosistem edukasi kripto — 6 layanan SvelteKit yang saling terhubung (situs publik, akademi, CMS admin dengan RBAC, REST API dengan dokumentasi Scalar, portal komunitas, media viewer) dengan aturan fondasi AI bersama lintas repo.

Dampak

Live di cryptosharia.id dengan e-learning lengkap, screening token, sertifikat, dan autentikasi aman JWT + Argon2id di seluruh stack microservice.

6 microservice E-learning + sertifikat Screening token Autentikasi diperkuat
