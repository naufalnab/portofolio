PT Adhi Karya
E-Risk — Enterprise Risk Management
Python Odoo 16 PostgreSQL REST API SSO
Problem

Enterprise risk processes were fragmented across spreadsheets and disconnected tools, with no real-time visibility into KPIs, treatments, or impact assessments.

Solution

Designed 30+ custom Odoo 16 modules — risk registers, interactive dashboards, KPI tracking, impact assessments, treatment realization, tier validation, SSO, and a public REST API.

Impact

Built to be a single source of truth for enterprise risk — with approval workflows, dark mode theming, and seamless integration into the PT. Adhi Karya platform.

PT Kripto Syariah Indonesia
CryptoSharia — Learning & Membership Ecosystem
SvelteKit TypeScript Tailwind 4 PostgreSQL Drizzle ORM OpenAPI 3.1
Problem

Indonesia lacked a credible Sharia-compliant crypto education platform with both e-learning depth and a verifiable token screening system.

Solution

Built a structured learning and membership platform for a crypto education ecosystem — 6 interconnected SvelteKit services (public site, academy, admin CMS with RBAC, REST API with Scalar docs, community portal, media viewer) with shared AI foundation rules across repos.

Impact

Live at cryptosharia.id with full e-learning, token screening, certificates, and JWT + Argon2id-secured auth across the entire microservice stack.

6 microservices E-learning + certs Token screening Hardened auth
