Layanan Komersial · Website

Pembuatan website yang siap kerja, bukan sekadar tampilan.

Saya bangun website yang langsung berfungsi sebagai mesin lead — terhubung WhatsApp, ter-tracking analytics, dan terstruktur agar tim Anda mudah update kontennya sendiri.

Basic
Rp 2.500.000 sekali bayar · 1 minggu

Untuk UMKM atau personal brand yang baru go-online.

  • 1 landing page (1 halaman, sampai 5 section)
  • Domain & hosting setup (.com / .my.id)
  • Mobile-responsive · SEO basic on-page
  • Form WhatsApp / kontak terhubung langsung
  • Revisi 2x dalam masa pengerjaan
Diskusi paket Basic
Paling populer
Pro
Rp 6.500.000 sekali bayar · 2-3 minggu

Untuk bisnis yang sudah jalan dan butuh website company profile.

  • Sampai 8 halaman (Home, About, Services, Blog, Kontak)
  • CMS sederhana untuk update konten sendiri
  • Integrasi WhatsApp, Google Analytics, Meta Pixel
  • SEO setup · sitemap · meta · schema.org
  • Revisi 3x · dukungan 30 hari setelah live
Diskusi paket Pro
Custom
Mulai Rp 15.000.000 kuotasi · 4+ minggu

Untuk web app, dashboard, e-commerce, atau sistem internal.

  • Custom UI/UX sesuai kebutuhan
  • Database & autentikasi (login, role, dashboard)
  • Integrasi payment, marketplace, atau API pihak ketiga
  • Deploy ke production · CI/CD · monitoring
  • Dokumentasi & handover

Berapa lama proses pengerjaan?
Basic 1 minggu, Pro 2-3 minggu, Custom mulai 4 minggu tergantung scope.
Apakah saya dapat source code?
Ya. Semua source code, akses domain, dan akses hosting menjadi milik Anda penuh.
Bagaimana sistem revisinya?
Revisi terbatas sesuai paket. Revisi major di luar scope dihitung terpisah dengan kuotasi transparan sebelum dieksekusi.
Apakah bisa update sendiri setelah jadi?
Paket Pro dan Custom dilengkapi CMS. Paket Basic landing page biasanya update via permintaan kecil.
Bagaimana cara pembayarannya?
DP 50% di awal, pelunasan saat website live. Untuk Custom bisa termin per milestone.
